== Description ==

Calculate how much fiat money your crypto portfolio is worth.

== Installation ==

1. Upload `my-cryptoo-portfolio.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your crypto portfolio in the widgets section
